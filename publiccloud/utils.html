lib/publiccloud/utils.pm

gcloud_install gcloud_install($url, $dir, $timeout)

This function is used to install the gcloud CLI for the GKE Google Cloud.

From $url we get the full package and install it in $dir local folder as a subdir of /root. Defaults are available for a simple call without parameters: gcloud_install()

fully_update_system

ssh_fully_patch_system($host);

Connect to the remote host $host using ssh and update the system by running zypper update twice. The first run will update the package manager, the second run will update the system. Transactional systems like SLE micro shall use transactional_update patch and reboot.
